# Student Management System

A console-based **Student Management System** developed using **Python** and Object-Oriented Programming (OOP).  
This project helps manage student records using a simple menu-driven interface.

---

## Project Overview

The Student Management System allows users to perform operations like adding, viewing, searching, updating, and deleting student records.  
All data is stored permanently using file handling.

---

## Features

- Add new student records
- View all students
- Search student by roll number
- Update student marks
- Delete student records
- Automatic data save and load
- Menu-driven console application

---

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling
- Dictionary Data Structure
